Code Example

TowTruck requires specific support for editors. There is active support for CodeMirror (used for example by Thimble), and ACE (used by Github, Cloud9).

ACE

function foo(items) {
  var i;
  for (i = 0; i < items.length; i++) {
    alert("Ace Rocks " + items[i]);
  }
}
