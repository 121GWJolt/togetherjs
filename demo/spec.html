Slowparse Error Reporting Specification

All of the following documentation is generated on-the-fly, in your browser, by Slowparse. If something looks amiss, run the actual output against the expected results at the test sute; if errors are reported, please feel free to file a bug.

You can also consult the live demo to see this error reporting used in an interactive widget.
