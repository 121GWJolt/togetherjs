Slowparse Error Reporting Specification

All of the following documentation is generated on-the-fly, in your browser, by Slowparse. If something looks amiss, it could be because there's a bug in Slowparse—please feel free to file a bug.

Once the API stabilizes, we'll take a snapshot of this page and doctest it to ensure that regressions don't occur.

You can also consult the live demo to see this error reporting used in an interactive widget.
