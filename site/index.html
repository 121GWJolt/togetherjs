Tour

Why use Connect?

By someone you trust.

Built and hosted by Mozilla, Connect is open source.

Create together.

Connect lets users communicate, co-author, co-browse and guide each other.

Built for the web.

Connect is implemented in Javascript; no software or plugins to install, and it's friendly with existing web pages, while still letting developers customize the experience.

How does Connect work?

Add Connect Javascript and HTML to your site.

Just copy and paste Connect's JS library and HTML snippet into your site or app to give it out-of-the-box collaboration tools and features.

Your site is now Connect enabled.

If your users want to collaborate with a friend, they can click the Connect button on your site.

Your users can then collaborate in real time!

Your users can send a unique Connect url to a friend. When the friend opens the link, each user sees the same page and can interact with it together in real time.

Features

Here's a list of built-in features you and your users will get right out of the box!

  • Video or Audio chat

    Connect uses Web RTC technology to enhance communication for your users.

  • User focus

    Your users can point to each other on a page or app.

  • User presence

    Connect enables your users to see each other in real time.

  • Text chat

    Your users can chat with each other with familiar instant messaging.

  • Co-browsing

    Your users can follow each other to different pages on the same domain.

  • Real time content sync

    Your users can see content on a site or app dynamically change together.

Try out Connect today!

Add it your site now in 2 easy steps!

Add the Javascript snippet

Simply copy and paste the TowTruck JavaScript snippet below into your site.

<script src="https://towtruck.mozillalabs.com/towtruck.js"></script>

Add the HTML button

After you've included the TowTruck JavaScript snippet, copy and paste this anywhere onto your site!

<button onclick="TowTruck(this); return false;">Start TowTruck</button>
